Добро пожаловать на мой сайт

Это простой шаблон с автоматическим деплоем через локальный GitlabCI. При каждом коммите в ветку main сайт обновляется сам.

Простота использования

Через nginx отдаются статические файлы, которые синхронизируются с репозиторием.
